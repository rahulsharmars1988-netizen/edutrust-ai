EduTrust AI Platform

A computer-implemented system for multi-dimensional outcome and trust evaluation in online higher education — using adaptive data processing, AI-powered prediction, and institutional credibility scoring.

System Overview

EduTrust AI is a patent-pending system comprising six integrated modules that work together to evaluate student outcomes, predict dropout risk, and generate trust and credibility indicators for online higher education institutions.

Architecture Layers

  • Data Acquisition Layer — Connects to LMS, SIS, examination systems, and student portals via REST API
  • Processing Engine — ETL pipeline for data normalisation and feature extraction
  • AI Module — Claude-powered prediction, analysis, and insight generation
  • Scoring Engine — Multi-dimensional composite outcome scoring
  • Trust & Credibility Module — Anomaly detection and credibility index computation
  • Output Interface — Dashboard, REST API, webhooks, and automated reports

Quick Start

1. Login

Navigate to index.html and sign in with your institution email. Use any email and a password of 6+ characters for demo access.

2. Connect Data Sources

Go to Settings → Data Source Connections and configure your LMS and SIS integrations.

3. Run Your First Analysis

Click "AI Analysis" in the sidebar and ask: "Which students are at highest dropout risk this semester?"

REST API

Authentication

POST /api/v1/auth/token Content-Type: application/json { "institution_id": "UNIV-001", "api_key": "et_live_xxxxxxxxxxxx" }

Get Outcome Scores

GET /api/v1/outcomes/scores?programme=all&period=semester Authorization: Bearer {token} Response: { "outcome_score": 78.4, "credibility_index": 91.2, "dropout_risk": 0.126, "trust_score": 84.7, "students_at_risk": 47, "anomalies_active": 3 }

Run Dropout Prediction

POST /api/v1/predict/dropout Authorization: Bearer {token} Content-Type: application/json { "student_id": "STU-2024-0847", "gpa": 2.1, "attendance_pct": 58, "lms_logins_per_week": 2, "assignments_missed": 5, "financial_aid": "partial", "forum_participation": "low" } Response: { "risk_score": 78, "risk_level": "High", "top_factors": [...], "interventions": [...], "confidence": 0.94 }
